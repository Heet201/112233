# Trueline Support Desk — 365 Ticket System

A comprehensive, responsive, and highly polished customer support and ticketing workspace built with **React**, **TypeScript**, and **Tailwind CSS**.

## ✨ Features

### 1. 🎫 Customer Portal
- **Create Tickets**: Submit tickets with rich metadata, including Category, Subcategory, Urgency/Priority level, and descriptions.
- **Message Threads**: Chat in real-time with support agents, add attachments, and view system logs.
- **Feedback & Rating**: Rate resolved tickets with direct feedback on the experience.

### 2. 🎧 Agent Workspace
- **Dynamic Assigned Queue**: Track, filter, and search open or pending tickets.
- **Live Response Feed**: Exchange replies with customers and update statuses (Open, In Progress, Pending, Resolved, Closed).
- **Control Panel**: Dynamically adjust ticket priority, reassign categories, and log notes.

### 3. 📊 Admin Dashboard & CRM
- **Executive Metrics**: Live count of active tickets, resolution times, client satisfaction ratings, and agent loads.
- **Visual Analytics**: Interactive breakdowns of tickets by priority, status, and category.
- **CRM Integration**: Contact lookup system monitoring active clients, company directories, and status logs.

---

## 🛠️ Tech Stack

- **Framework**: React 19 (Functional Components & Hooks)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (Fully responsive, custom modern layout)
- **Animations**: Motion (framer-motion) for fluid transitions
- **Icons**: Lucide React
